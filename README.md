# Project AI Manager

AI-powered project management assistant that breaks down projects into tasks and estimates timelines.

## Features
- Natural language project breakdown
- Timeline estimation with confidence intervals
- Dependency detection
- Risk identification
- Integration with Jira, Linear, GitHub

## Installation
```
pip install -r requirements.txt
```

## Usage
```
python main.py plan 'Build a mobile app with auth and payments'
python main.py estimate project.json
```
